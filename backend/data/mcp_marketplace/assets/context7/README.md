# Context7 (MCP Server)

Context7 provides up-to-date documentation and code examples to your agent. This MCP server exposes tools for searching and retrieving library documentation.

## Install Options

- NPX: requires `node` and `npx`.
- HTTP (Remote): connect to Context7 hosted MCP endpoint.

## Authentication

- NPX (Local): does not require an API key.
- HTTP (Remote): requires a Context7 API key (sent via `Authorization: Bearer ...`).
