# Fetch (MCP Server)

Fetch is an MCP server that fetches web pages and converts HTML to Markdown for downstream processing.

## Install Options

- UVX (recommended): requires `uv` with `uvx` available on PATH.
- Python module: requires `mcp-server-fetch` installed in your Python environment.

## Notes

- If you are behind a proxy, configure `HTTP_PROXY` / `HTTPS_PROXY` during installation.
