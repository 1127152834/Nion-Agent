# Frontend Workbench

`Frontend Workbench` 用于在聊天页右侧插件插槽中完成前端项目的浏览、编辑、构建与预览。

## 功能亮点

- 文件树与编辑器联动（支持快速打开与保存）
- 构建与启动分离，确保“先构建再启动”
- 预览与日志双面板，便于定位问题
- 插槽模式渲染，视觉框架与“工作目录”保持一致

## 使用方式

1. 在“插件市场”安装该插件，或下载 `.nwp` 后本地安装。
2. 打开聊天页右侧面板，切换到“插件”模式。
3. 选择 `Frontend Workbench` 插件并开始操作项目。
