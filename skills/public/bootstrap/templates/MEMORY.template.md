# MEMORY.template.md

用于生成本次 bootstrap 的“记忆初始化清单”（不会写成文件，而是通过 `setup_agent(memory_items=...)` 写入 OpenViking Memory）。

默认使用**本次对话语言**。只记录“稳定且对该智能体长期有用”的信息，避免敏感信息与一次性信息。

tier 只允许：
- `profile`：该 agent 需要长期记住的上下文事实（项目背景、术语、约束、固定流程）
- `preference`：该 agent 的协作偏好（输出结构、默认语言、pushback 强度等）

输出格式（用于工具调用时直接复制）：

```json
[
  { "tier": "profile", "content": "（一条稳定事实/上下文；一句话；只含 1 个信息点）" },
  { "tier": "preference", "content": "（一条协作偏好；一句话；只含 1 个偏好）", "confidence": 0.8 }
]
```

## 规则

1. 3–8 条为宜，宁缺毋滥。
2. 每条只表达 1 个事实/偏好，不要在一条里塞多个分号。
3. 避免高度敏感信息（身份证号、住址、联系方式、财务账号等）与用户不愿长期保存的内容。
4. 如果用户明确不希望写入记忆：输出空数组 `[]`，并在落盘时不要传 `memory_items`。
