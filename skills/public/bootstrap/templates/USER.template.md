# USER.template.md

用于生成“全局用户画像”内容（会写入根目录 `USER.md` 的一个 marker 块内）。该内容会被注入到系统提示词的 `<user-profile>` 区块，用于长期个性化。

默认使用**本次对话语言**。只记录稳定偏好与长期信息，避免敏感细节与一次性信息。

注意：不要在内容里包含任何 marker（marker 由后端工具自动包裹与幂等更新）。

```markdown
# 用户画像

## 背景（可选）
- [职业/领域/上下文：只写用户愿意沉淀到全局的部分]

## 目标与优先级（可选）
- [长期目标]
- [近期优先事项]

## 沟通偏好
- 语言：[默认语言；术语偏好；是否允许中英混排]
- 结构：[偏好 TL;DR / 要点列表 / 详细推导]
- 节奏：[先问后做 vs 先给草案]
- Pushback：[希望直接指出问题还是委婉提醒]

## 禁区与雷区
- [不希望被怎么对待/不希望出现的行为]
- [涉及金钱/外联/高风险动作时的默认规则：必须先征求同意等]
```

## 规则

1. 只写“稳定且有用”的信息，宁缺毋滥。
2. 避免写入敏感个人信息（身份证号、住址、联系方式、财务账号等）或任何用户不愿长期保存的内容。
